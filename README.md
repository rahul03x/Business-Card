# 💼 Business Card

A modern and responsive digital business card built using **HTML5** and **CSS3**. This project features a clean and elegant layout that showcases personal information, contact details, and social media links.

## ✨ Features

* Clean and modern business card design
* Responsive layout
* Company logo section
* Name, designation, and company details
* Contact information (Phone, Email, Address)
* Social media icons with hover effects

## 🛠️ Technologies Used

* **HTML5** 
* **CSS3** 
* **Visual Studio Code (VS Code)**
* **Boxicons** 

## 🎨 Customization

You can easily personalize the business card by:

* Replacing the logo with your own
* Updating your name, designation, and company
* Adding your contact details
* Editing the color scheme in style.css
